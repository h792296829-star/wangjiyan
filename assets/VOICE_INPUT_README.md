# 公务员培训助手 - 语音输入功能使用说明

## 功能介绍

公务员培训助手现已支持**语音输入**功能，让您可以通过语音与助手进行交互，更加便捷地使用各项功能。

## 如何使用

### 1. 访问界面

服务启动后，在浏览器中访问：
```
http://localhost:5000/
```

### 2. 语音输入

#### 方法一：点击语音按钮
- 点击输入框旁边的麦克风图标 🎤
- 开始说话
- 系统会自动识别您的语音并转换为文字
- 点击发送按钮或按 Enter 键发送

#### 方法二：文字输入
- 直接在输入框中输入文字
- 点击发送按钮或按 Enter 键发送

### 3. 快捷操作

界面提供了三个快捷按钮：
- **出一个组织管理题**：快速生成组织管理类题目
- **出一个情景模拟题**：快速生成情景模拟类题目
- **给答案**：获取当前题目的答案

## 支持的功能

### 出题功能
说出以下内容，让助手生成题目：
- "出一个组织管理题"
- "出一个综合分析题"
- "出一个情景模拟题"
- "出一个人际沟通题"
- "出一个应急应变题"
- "出一个职业认知题"
- "出一个自我认知题"

### 答题功能
看完题目后，说出以下内容获取答案：
- "给答案"
- "答题"
- "解析"

### 讲解功能
直接说出您想了解的主题，助手会进行详细讲解：
- "基层治理创新"
- "政务服务数字化"
- "社区养老服务"
- 等等

## 浏览器兼容性

语音输入功能需要浏览器支持 **Web Speech API**。

### 支持的浏览器
- ✅ Chrome（推荐）
- ✅ Edge
- ✅ Safari
- ✅ Opera

### 不支持的浏览器
- ❌ Firefox（部分版本）
- ❌ IE（任何版本）

如果您的浏览器不支持语音识别，可以使用文字输入功能。

## 注意事项

1. **首次使用**：首次使用语音功能时，浏览器会请求麦克风权限，请点击"允许"。
2. **网络要求**：语音识别需要网络连接，建议在稳定的网络环境下使用。
3. **清晰发音**：说话时请清晰、准确地发音，以提高识别准确率。
4. **环境噪音**：在安静的环境中使用，避免背景噪音干扰。

## 技术特点

### 前端技术
- **语音识别**：使用浏览器原生的 Web Speech API
- **实时反馈**：显示语音识别状态和中间结果
- **流式响应**：实时接收和显示助手的回复
- **响应式设计**：支持桌面和移动设备

### 后端技术
- **FastAPI**：提供高性能的 API 服务
- **SSE 流式传输**：支持流式输出，提升用户体验
- **LangChain**：智能对话框架

## 常见问题

### Q: 为什么看不到麦克风按钮？
A: 麦克风按钮应该始终显示，如果您的浏览器不支持语音识别，按钮会显示为灰色（禁用状态）。点击它会弹出提示信息。建议：
1. 检查浏览器是否为 Chrome、Edge 或 Safari
2. 点击右上角的"浏览器兼容性测试"按钮进行检测
3. 如果浏览器不支持，请切换到 Chrome 或 Edge 浏览器

### Q: 为什么点击语音按钮没有反应？
A: 请检查：
1. 浏览器是否支持 Web Speech API
2. 是否授予了麦克风权限
3. 是否有其他程序占用了麦克风
4. 点击"浏览器兼容性测试"按钮检查浏览器支持情况

### Q: 麦克风按钮显示为灰色怎么办？
A: 这表示您的浏览器不支持语音识别功能。解决方法：
1. 使用 Chrome、Edge 或 Safari 浏览器
2. 或者直接使用文字输入功能（不受影响）

### Q: 如何测试我的浏览器是否支持？
A: 点击页面右上角的"🔍 浏览器兼容性测试"按钮，系统会自动检测您的浏览器支持情况。

### Q: 语音识别不准确怎么办？
A: 建议：
1. 在安静的环境中使用
2. 清晰、准确地发音
3. 避免使用口语化表达

### Q: 可以连续说话吗？
A: 当前版本每次录音最多识别一次完整的表达。如需发送多条消息，请多次点击语音按钮。

### Q: 如何停止录音？
A: 再次点击语音按钮即可停止录音。
